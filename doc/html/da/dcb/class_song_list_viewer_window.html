Detailed Description

Allows the user to choose which songs to import from a folder.

This class implements event handling for a window that displays lists of songs. The window has different modes that it can be run in, and these modes dictate the actions that are available for the user in the window.

void SongListViewerWindow::updateSongListFromTable ( )
private

Updates the songs with edits made in the dialog's table.

When the user edits an entry in the Song list, a struct is used to keep track of what changed. This function goes through all of the marked edits and updates the songs in the list. This function is only called right before the window is closed if the user clicked the Ok button and either edits occurred or songs were imported. We only edit the Song list in this function because if the edits were applied to the Song at the time when the user makes them, these edits would be kept even if the window was cancelled.
