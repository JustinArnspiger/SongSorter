Detailed Description

Represents a Song.

This class is a container that represents a Song according to its metadata (artist, album, etc.) as well as its overall rank and file path.

[static] Song *Song::createSongFromFilePath(QString aFilePath)

Returns a new Song whose metadata is retrieved from the file path. aFilePath The given file path to the song.

This function is for when a song's metadata is unable to be retrieved through the standard method of using QMediaPlayer. In order to try to approximate the metadata, this function will use parts of the file path with the assumption that if the file is something like example_song.mp3, then the song's name is Example Song.

The function also tries to obtain the artist and album by assuming (if applicable) that the file path is formatted as .../artistName/albumName/song.<file_extension>.

Note: This function will allocate a new song, so the song that is created must be deleted later.
