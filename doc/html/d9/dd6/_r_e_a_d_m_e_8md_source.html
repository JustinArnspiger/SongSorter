1 # Song Sorter
2 This program is designed to make it easy for people to sort & rank their favorite songs, artists, and albums. It will accomplish this by first getting a list of songs from the user and then presenting to the user a pair of songs and having them choose which one of the two they like more. The program will then continue to present pairs of songs to the user until each song is sorted.

7 ## Planned features
8 * Efficient Sorting
9  * The program will try to reduce the number of comparisons that the user has to make as much as possible. It will attempt to do this by using a self-balancing binary tree to store the results of each comparison.
10 * Media Player Functionality
11  * When the user is prompted to compare songs, they will have the option of listening to them so that they can make a more accurate decision about which one they prefer.
12 * Categorization of Results
13  * The user will have the option of having the results grouped into categories such that they will be able to see how each group of songs compares to one another. There will also be a variety of different ways in which the groups can be sorted.
14  * Planned Categories:
15  * Album
16  * Artist
17  * Genre
18  * Year
19  * Decade
20  * Planned Sorting Methods:
21  * Average Rank
22  * Median Rank
23  * Highest/Lowest Rank of Individual Song
24  * All of the above with the ability to only take into account the highest-ranked or lowest-ranked half of songs in the group.
25 * Ability to Export Results
26  * The user will be able to export the results of the sorting to a csv file. This option will be available for both grouped and ungrouped results.
27 * Playlist Generation
28  * The user will also be able to generate playlists using the results of the sorting.
